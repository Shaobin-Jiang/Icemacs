## Installation

```bash
git clone https://github.com/Shaobin-Jiang/Icemacs ~/.emacs.d
```

Launch emacs, wait for the necessary packages to finish installing, and press <kbd>M-x</kbd> run these commands:

```elisp
treesit-auto-install-all
nerd-icons-install-fonts
```
